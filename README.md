# Text Analyzer

Веб-приложение для анализа текстов на русском и английском языках, разработанное на Python с использованием Streamlit.

# О проекте

Text Analyzer позволяет загружать текстовые файлы и получать основную статистику по тексту, включая количество слов, предложений, символов, частотность слов и облако слов.

Проект был создан в рамках учебной практики для закрепления навыков обработки текста, работы с библиотеками Python и создания веб-интерфейсов.

--- 

## Возможности

* Автоматическое определение языка текста (русский или английский)
* Подсчёт количества слов
* Подсчёт количества предложений
* Подсчёт количества символов
* Расчёт средней длины слова
* Удаление стоп-слов
* Подсчёт частоты слов
* Вывод топ-10 самых популярных слов
* Генерация облака слов (Word Cloud)
* Загрузка текстовых файлов формата TXT

---

## Используемые технологии

* Python

* NLTK

## Установка

### Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/text-analyzer.git
```
### Перейдите в папку проекта:
```bash
cd text-analyzer
```
### Установите зависимости:
```bash
pip install -r requirements.txt
```
---

### Запуск
```bash
streamlit run app.py
```
После запуска приложение откроется в браузере.

---

## Пример работы

1. Загрузите TXT-файл.
2. Приложение автоматически определит язык текста.
3. Получите статистику:
    * количество слов;
    * количество предложений;
    * количество символов;
    * среднюю длину слова.
4. Просмотрите топ-10 наиболее часто встречающихся слов.
5. Постройте облако слов.
